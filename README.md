## My Year in Projects

> *Everything below was built this year, from scratch, while taking full-time engineering courses.*

### 👗 StyleAI — AI Personal Stylist
**The most complex thing I've built.**

Upload your clothes → FashionCLIP analyzes them → Claude recommends outfits → Weather API adapts suggestions to today's forecast.

Full-stack: FastAPI backend with SQLAlchemy ORM, Next.js frontend, Qdrant vector database, real AI inference pipeline.

```
Stack: FastAPI · FashionCLIP · Claude API · Next.js · SQLite/SQLAlchemy · Qdrant
What I learned: AI integration, vector DBs, full-stack architecture, REST API design
```

[![StyleAI](https://img.shields.io/badge/View_Repo-StyleAI-7c3aed?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Nour-Faker/StyleAI)

### 🍅 AI Pomodoro Forecaster — Focus Intelligence
**Time series forecasting applied to personal productivity.**

Built a full Pomodoro timer with Python/Tkinter — sessions, streaks, habits, goals. Then integrated Google's TimesFM model to predict your optimal study schedule based on your past focus patterns. The AI answers natural language questions about your productivity.

```
Stack: Python · Tkinter · Google TimesFM · Custom AI module
What I learned: Time series forecasting, desktop app architecture, data persistence
```

[![Pomodoro](https://img.shields.io/badge/View_Repo-AI--Pomodoro-7c3aed?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Nour-Faker/AI-Pomodoro-Forecaster)

---

### ✋ HandGesture Control — Computer Vision Mouse
**No hardware. Just your webcam and hand.**

Real-time hand tracking that replaces your mouse entirely. MediaPipe detects 21 hand landmarks per frame, custom gesture logic maps finger positions to cursor events, OpenCV handles the video pipeline at low latency.

```
Stack: Python · OpenCV · MediaPipe · PyAutoGUI
What I learned: Real-time CV pipelines, landmark detection, gesture state machines
```

[![HandGesture](https://img.shields.io/badge/View_Repo-HandGesture--Control-7c3aed?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Nour-Faker/HandGesture-Control)

---

### 🚑 SmartEMS Tunisia — Emergency Dispatch System
**Algorithms that matter when time means lives.**

Intelligent emergency dispatch and routing system for Tunis. Implements A* pathfinding for real-time optimal routing between ambulance units and emergency sites, with intelligent dispatching logic to assign the nearest available unit.

```
Stack: Python · A* Algorithm · Graph theory · Routing logic
What I learned: Pathfinding algorithms, graph data structures, system design
```

[![SmartEMS](https://img.shields.io/badge/View_Repo-SmartEMS--Tunisia-7c3aed?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Nour-Faker/SmartEMS-Tunisia)
